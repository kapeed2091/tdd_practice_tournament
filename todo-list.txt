Usecases:
    - User can create a knock-out tournament with configurable number of rounds and start time
    - User can get list of all tournaments
    - User can subscribe to a tournament which is not yet started
    - User can play matches once the tournament is started
        - User will submit his score in the match
    - User can progress to next round if he wins the match
    - User can get his current round number
    - User can get his opponent user profile in any round
    - User can get the winner user profile of the tournament

Assumptions:
    - Tournament will always be full before start time
    - All players will play all their matches
    - In case of tie, Whoever submits the score first is winner
    - User profile has Name, Age, Gender
